Convai Character Customization

Creating Your Own Character

  1. Visit Convai.com and sign in to your account.
  2. Click the "Create Character" button on your Dashboard.
  3. Set up your character's basic details:
    • Name your character
    • Select a voice from available options
    • Write a detailed backstory
  4. Enhance character personality (Optional):
    • Select from preset speaking styles or create custom ones
    • Adjust personality traits (openness, meticulousness, extroversion, etc.)
    • Add catchphrases and examples
  5. Add a knowledge bank (Optional):
    • Upload text files (up to 1MB total)
    • Add specific knowledge your character should have

Using Custom Characters in the App

  1. After creating your character, you'll find a unique Character ID in the character info section for that character
  2. Copy the Character ID and paste it into the "Character ID" field after clicking on the edit button for that character in the desktop pet app

Animation Settings

Default Frame Duration: Controls how fast the character's animation frames change (in milliseconds). Lower values create faster animations.

Default Move Duration: Determines how quickly the character moves across the screen (in milliseconds). Lower values result in faster movement.
